Installation

First, clone or fork this repo and download the source code to your local computer.

Second, set up a Firestore database. You can follow the Firestore Setup Cheat Sheet.

Third, you will want to initialize your Firestore database with example data. There is example data in the assets/initialData directory. The most convenient way we know to upload JSON files to Firestore is through Firefoo. For further information, see these instructions for uploading data to Firestore with Firefoo.

Important note: when initializing the database with sample data, you must tell Firestore to use the "id" field in the sample data as the documentID. The system code depends upon that equivalency.
